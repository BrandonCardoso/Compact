General Information

This is the only place where I make my skin available. Any mirrors or copies found elsewhere are not endorsed by me.

If you like to live on the edge I might sometimes have a dev branch going for any Steam beta development. This is only when I have the time and inclination for it though. I will always post what Steam version my skin is compatible with, so please check that before reporting any issues.
